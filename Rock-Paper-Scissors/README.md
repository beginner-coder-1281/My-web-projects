# Overview
This rock paper scissors game provides a easy to play game with accurate winner selection and displaying of choices
# Features
• A button for rock, paper and scissors to get user choice\
• Random computer choice generation\
• A try again button\
• Tracking winner 
# Tools used
• HTML for the structure\
• CSS for the designs\
• Javascript for generating computer choice and checking the winner, as well as handling user choice
