# Overview
A simple minimalistic stopwatch in order to track time accurately and display it
# Features
• A start, stop and reset button\
• Tracking of time using Date.now()\
• Tracking of time even if the tab isn't running
# Tools used
• HTML for the basic structure\
• CSS for the designs\
• Javascript for Tracking the time and handling button pressing
