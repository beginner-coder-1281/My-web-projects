# Overview
This is a todo list app which provides users with a functional and visibly pleasing experience.
# Features
• Ability to add and remove tasks\
• Ability to mark tasks as done and revert the change if needed\
• Saving of tasks to local storage as well as their done/not done status
# Tools used
• HTML for the task input and add button as well as the tasks container\
• CSS for all the designs and the marking as done feature\
• Javascript for adding tasks
