# Overview
This project presents a quiz to the user with saving user choices, generating questions and allowing travel between questions through the "next" and "previous" buttons, allowing users to test their knowledge and get their scores at the end as well as the correct answers
# Features
• Dynamic questions generation\
• Saving user selections\
• Next and previous buttons for navigation\
• Handling errors properly
# Tools used
• HTML for the next and previous buttons\
• CSS for the designs\
• Javascript for generating the questions and handling the submission of the quiz
# Note to contributors
To add questions to the quiz, you can add the question in the questions array, as well as the options and answer and the question will be automatically generated.
