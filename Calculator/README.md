# Overview
This calculator helps users evaluate mathematical equations in a effortless way, when the user types a mathematical expression, they can press the equal to button in order to get the answer
# Features
• Addition\
• Subtraction. 
• Multiplication. 
• Division. 
• Square
# Tools used
• HTML for the basic structure 
• CSS for the designs
• Javascript for the input validation and calculation
• math.js module for evaluating the expressions
