# Overview
This is a number guessing game which chooses a random number based on the difficulty chosen by the user and allows the user to guess the random number in order to win, allowing for a fun gaming experience.
# Features
• Difficulty selection menu\
• Random number generation\
• Attempts tracking\
• Helpful hints and feedback\
• Handling Invalid input properly
# Tools used
• HTML for the basic structure\
• CSS for the designs\
• Javascript for the game logic
